## 概述

API 请求失败时，服务端会返回一个 JSON 格式的响应体。它可以用于定位具体错误原因。如：
```
{"msg": "file or directory not found", "code": 40400001, "id": "11e4a729d24943409933a5e8314cd32e"}
```

** 字段说明 **

* `code`：错误码。具体含义见 [API 错误码表](#api)。
* `msg`：错误的简单描述。
* `id`：又拍云为该请求生成的唯一 ID。
